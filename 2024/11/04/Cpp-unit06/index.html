指针和const

指向常量的指针不能用于改变其所指的对象的值。要想存放常量对象的地址，只能使用指向常量的指针。

const 指针

常量指针必须初始化，初始化完成，则它的值就不能改变。把*放在const关键字之前用以说明指针是一个常量，这样的书写形式隐含着一层意味，即不变的是指针本身的值而非指向的那个值。

！！！重要区别

顶层const、底层const

贴一段英文：As we’ve seen, a pointer is an object that can point to a different object. As a result,

we can talk independently about whether a pointer is const and whether the objects

to which it can point are const. We use the term top-level const to indicate that the

pointer itself is a const. When a pointer can point to a const object, we refer to

that const as a low-level const.

top-level const 表示指针本身是个常量，low-level const 表示指针所指的对象是一个常量。

举几个例子

声明的时候：const int 和 int const 是等价的 。声明指向常量的指针是底层const

来一个助记：如果 const 右结合修饰为 类型 或者 * ，那这个 const 就是一个底层 const； 如果const 右结合修饰为标识符，那这个const 就是一个顶层 const。

代码解析：ci是const，这个值不能修改； p是常量指针，指向整数类型的变量i。由于指针是const，所以p本身的值（即指向的地址）不能被修改，但可以通过p修改其指向的值。

*p = 0 可以通过指针p修改i的值，将其设置为0 但是不能修改p指向的地址
